# 🧮 Modern UI Calculator

## 📌 Overview
This is a simple but modern calculator built using HTML, CSS, and JavaScript.  
The goal of this project was to improve my JavaScript logic handling and UI design skills.

---

## ✨ Features
- Clean and modern UI design
- Basic arithmetic operations (+, −, ×, ÷)
- Responsive layout (mobile + desktop)
- Button click interaction
- Keyboard support (optional if you added it)
- Smooth and intuitive user experience

---

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
